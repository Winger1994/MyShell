# MyShell
SJTU CS356 and SJTU JI VE482 Course Project
Implement a simple Unix shell

## Implemented
* pipe
* file redirect
* pipe: multiple pipes
* file redirect: continue input until filename is given
* interrupt (TODO)

## Not Implemented
* flow control (diffcult)

## Deployment
* Unix-like System (Linux or macOS)
* make
* ./MyShell
